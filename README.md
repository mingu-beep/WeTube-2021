## 구글 크롬 Err_Connection_Refused 오류 해결법

1. 명령 프롬프트를 관리자 권한으로 실행한다.
2. 'netsh winsock reset catalog'를 입력한다
   - 카탈로그 재설정 문구를 확인합니다.
3. 'ipconfig /flushdns'를 입력한다
   - DNS 확인자 캐시를 플러시 했습니다.

- 계속 같은 현상이 반복되서 크롬의 보안 정책상 문제라 판단하고 브라우저를 바꿨다.
